# Day 1 - Database Fundamentals

## What is Data?
Data is a collection of raw facts and figures that have not been processed.

Example:
101, Varsha, 85

## What is Information?
Information is processed data that has meaning and is useful.

Example:
Varsha scored 85 marks in SQL.

## What is Database?
A database is an organized collection of related data that can be easily stored, managed, and accessed.

## What is DBMS?
DBMS (Database Management System) is software used to create, store, update, manipulateand retrieve data.

Examples:
- MySQL
- PostgreSQL
- Oracle Database

## What is RDBMS?
RDBMS (Relational Database Management System) stores data in tables and maintains relationships between tables.

## DBMS vs RDBMS

DBMS:
- Stores data
- Relationships are not mandatory
- Less suitable for large datasets

RDBMS:
- Stores data in tables
- Maintains relationships using keys
- Better for large datasets

## Real-Life Example
