## Key Points

- ORDER BY sorts the output of a SELECT query.
- ASC is the default sorting order.
- ASC sorts:
  - Numbers: Lowest → Highest
  - Text: A → Z
  - Dates: Oldest → Newest
- DESC sorts:
  - Numbers: Highest → Lowest
  - Text: Z → A
  - Dates: Newest → Oldest
- Multiple columns can be sorted together.
- WHERE filters rows before ORDER BY sorts them.

## SQL Execution Order

1. FROM
2. WHERE
3. SELECT
4. ORDER BY
